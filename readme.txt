== Description ==

Zedna 301 Redirects provides an easy method of redirecting requests to another page on your site or elsewhere on the web.
Supports import & export, wildcards, bulk removal. Plugin also works with GET requests like UTM tags.

== Installation ==

1. Upload Zedna 301 Redirects to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add redirects on the Settings > 301 Redirects page.
